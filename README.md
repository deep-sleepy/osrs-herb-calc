## Features

- **Live GE prices** from the OSRS Wiki real-time API (same source as RuneLite/Flipping Utilities) — 5-minute volume-weighted averages so prices are stable and consistent
- **Auto-refreshes every 5 minutes** with a countdown timer
- **All 10 herb patches** individually toggleable — disease-free patches (Weiss, Trollheim, Hosidius, Harmony, Civitas illa Fortis) correctly apply 0% death rate
- **Diary bonuses** — Kandarin diary (Medium/Hard/Elite) for Catherby, Hard Kourend diary for Hosidius and Farming Guild
- **Resurrect Crops** — shows Worth it / Not worth per herb based on live rune costs vs seed cost × your resurrect chance at your magic level
- **Sell as Grimy, Clean, or Unfinished potion** — vial of water cost automatically deducted for unfinished potions
- **Per-patch breakdown tables** — expandable tables showing profit and XP for each patch group separately so you can see exactly which herbs are worth it at protected vs unprotected patches
- **Sortable columns** — click any column header in the main table or any breakdown table
- **Remembers your settings** — farming level, magic level, compost, patches, diaries, equipment all saved in your browser and restored next visit
- **Mobile friendly** — collapsible settings panel, touch-friendly controls, works in any mobile browser

## How to use

Just open the link and bookmark it. Settings save automatically so everything will be exactly as you left it next time.
